# DevRoast

## Tech Stack
- Next.js 16 (App Router)
- TypeScript
- Tailwind CSS v4
- Biome (lint/format)
- Base UI (primitives)

## Variáveis de Design
Cores e configurações em `src/app/globals.css` (@theme).

## Padrões de Componentes
Ver `src/components/ui/AGENTS.md`.

## Estrutura
- `src/app` - rotas e páginas
- `src/components/ui` - componentes UI
- `src/lib` - utilitários e dados

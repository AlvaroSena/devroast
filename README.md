# DevRoast

Paste your code. Get roasted.

DevRoast is a web app that lets you submit your code and receive brutally honest feedback. Think of it as a code review from that one senior developer who doesn't sugarcoat things.

## Features

- **Submit Code** - Paste your code and get it roasted
- **Roast Mode** - Toggle between honest feedback or full roast mode
- **Leaderboard** - See the worst code ranked by shame
- **Syntax Highlighting** - Beautiful code display with Shiki

## Tech Stack

- Next.js
- React
- Tailwind CSS
- Base UI
- Shiki

## Getting Started

```bash
# Install dependencies
pnpm install

# Start development server
pnpm dev

# Build for production
pnpm build
```
